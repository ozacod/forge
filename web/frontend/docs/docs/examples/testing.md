# Testing & Benchmarking

## Testing
Run tests generated by cpx (GoogleTest, Catch2, or doctest).
```bash
cpx test          # all tests
cpx test -v       # verbose
cpx test --filter MySuite.*
```

## Benchmarking
Run benchmarks (Google Benchmark, Catch2, or nanobench).
```bash
cpx bench            # build + run benchmarks
cpx bench --verbose  # show build output
```

Keep tests close to code and run them before pushes; hooks can enforce this via `cpx hooks install`.
